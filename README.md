# Student Marks Data Analysis

This is a beginner data analysis project built using Python.

## Project Description
This project analyzes student marks and provides useful insights such as:
- Average marks
- Highest marks
- Lowest marks
- Top student
- Data visualization using a bar chart

## Technologies Used
- Python
- Pandas
- Matplotlib

## Files in the Project
- analysis.py : Python script for data analysis
- students.csv : Dataset containing student marks

## Output
The program calculates statistics and displays a bar chart showing student marks.
